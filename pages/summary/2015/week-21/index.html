C++03以前ではオーバーロードされたコンストラクタが呼び出せない

以下のように、コンストラクタから別のコンストラクタを呼び出すことで、
処理を共通化するのはわりと自然ではないかと思います。

これはdelegate constructorとうい名前がついており、
残念ながらC++11以降じゃないと使えません。

struct A {
  A() : A(0) {}
  A(int num) : a(num) {}
  int a;
};
